## 소개

**TypeScript, React/Next.js 기반 프론트엔드 개발자 (3년 경력)**

복잡한 사용자 인터페이스를 직관적이고 성능 최적화된 웹 애플리케이션으로 구현하는 것을 전문으로 합니다. Vanilla JavaScript부터 현대 프론트엔드 기술 스택까지 깊이 있는 이해를 바탕으로, 레거시 시스템 개선과 신규 서비스 개발에서 검증된 성과를 만들어왔습니다.

**핵심 전문성:**

- **WYSIWYG 에디터 개발**: 복잡한 편집 인터페이스를 메모리 효율적이고 직관적인 사용자 경험으로 구현
- **성능 최적화**: DOM 조작 최적화와 이벤트 위임을 통한 대규모 애플리케이션 성능 개선
- **모듈 아키텍처 설계**: 확장 가능하고 유지보수성 높은 프론트엔드 시스템 구축
- **현대 프론트엔드 기술**: TypeScript, React, Next.js를 활용한 타입 안전하고 확장 가능한 웹 애플리케이션 개발
